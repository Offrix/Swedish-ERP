# @swedish-erp/domain-travel

Travel engine for FAS 9.2:

- trip object with per-day domestic and foreign allowance valuation
- official 2026 domestic and foreign normal amount support
- meal reduction, long-stay reduction and night allowance handling
- mileage reimbursement split into tax-free and taxable parts
- expense receipt reimbursement and travel advance settlement payloads
- payroll line payload generation for traktamente, mileage, expenses and advance deduction
