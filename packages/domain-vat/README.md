# @swedish-erp/domain-vat

VAT decision boundary skeleton with rule-pack driven classification and declaration mapping.

## Phase 4.1 scope

- VAT code masterdata with declaration box and booking-template bindings.
- Versioned VAT rule packs resolved by jurisdiction and effective date.
- Explainable VAT decision objects with input hash, rule-pack trace and warnings.
- Manual review queue when mandatory VAT facts are missing or a candidate conflicts with structural facts.
