# @swedish-erp/domain-projects

Project engine for FAS 10.1 and 10.3:

- project profiles with billing model, revenue recognition model and project-reference aliasing
- versioned project budgets with cost and revenue lines per reporting period
- resource allocations with planned minutes plus bill- and cost-rate support
- actual-cost, WIP and forecast snapshots sourced from time, payroll and AR
- deterministic project margin and forecast-at-completion calculations
- project change orders for ATA with explicit status transitions and customer-approval handling
- project build-VAT assessments that wrap the VAT engine for Swedish construction reverse-charge scenarios
- audit trail for budget approval, resource planning, ATA transitions and build-VAT decisions
