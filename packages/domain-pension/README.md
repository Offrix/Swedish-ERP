# @swedish-erp/domain-pension

Pension engine for FAS 9.3:

- ITP1, ITP2, Fora and extra pension enrollment catalogs with date-effective provider mapping
- salary exchange simulation with threshold warning, employer markup and pause/resume-safe agreement state
- pension basis snapshots and pension events per reporting period
- payroll line payload generation for salary exchange deduction, pension premiums and special payroll tax
- provider report generation for Collectum, Fora and custom supplementary pension flows
- invoice reconciliation support with deterministic audit trail
